This section contains my thought process on the technical aspects, aesthetics and the accessibility aspects


Technical

I followed the guide on the UTS specification and included the list of the site structure made into a table.

File Name Title File Type Description
index.html Home HTML Introduction and home screen of the website
past.html Past HTML A brief timeline of my life
future.html Future HTML A quick description of what I want to do in the future
comments.html Comments CSS Technical aspect of this website
websystems.css Title HTML Styling that changes the looks of the website


For every section of the website, I used the body, main and then div as its structure. I have also used padding and line-height so that they website doesn't look squished together.
Let's start off with the index.html. On the Header I made the title of "Web Systems Assignment" and also the navigation section which included; "Home", "Past", "Future" and "Comments". The navigation section were made with unordered lists and lists. Inside the lists I put in the a tags and also href so I would be able to link the navigation bar.


On the Main section, I put all my information there for the readers to read. I mainly used <div> tags to section and also edit my work with CSS after putting classes on my divs. The tags I have included in the main section of this assignment were heading tags (<h2>, <h3>, <h4>), paragraphs (<p>), lists (<ul>, <li>) and images (<img>). These were the basic layouts on all the sections for this website although some additions can be seen on the Home and Past section.

For the Home section,<ul> and <li> tags were used to show the readers what additional subjects I was taking this semester for uni. When clicked, the links would take them to another tab instead of the page they were on by using the target ="_blank" attribute.

On the Past section, I have created a little timeline by creating a few <div> tags and having the year, school/ work, and a small description about it. I then created a small line on the side with 'border-left: 2px solid' to create a long line down on the left hand side to make it look like a timeline going down.


Finally for the Footer I made a simple look with just using one <p> tag and using a <div> before it with a class called 'footer__desc'. This was to change the color to black and notify the reader that it was the end of the page. Included inside it was my name, my university and also my student ID.


Aesthetics

For me, I have chosen a simple design for this website as I did not want an explosion of colors for the people that view my website. I decided to go for a minimalistic look with only using 2 main colors throughout this website.


Colors

In my thought process, I decided to have a background image for my website, unfortunately it was hard for me to find a good color for my texts and seemed like it was taking over the website with how many colors there were. I then decided to make the background black and have the text white, but the colors felt too sharp for my eyes. Lastly I decided to use a beige like color and a lighter shade of black for my website so that It would be pleasing to the eyes.

Font

I have used a few different font sizes in this website to signify that they are either a title, a new topic or a new paragraph. For the font style, I used 'Rubik' as its text looks 'nicely flowing' for this website. For users that might not have access to the font, I have also put in 'sans-serif' as a backup font. For me a well chosen font is as important as color when it comes to designing a website. If I had chosen 'Times New Romans' as my font, It would look like my website was made in the early 2000's and would not fit in my aesthetics of this website.


Interaction

For the Interaction aspect in this website, I have made a few links where users can click on. One of them is in the 'Home' page. The subjects that I have stated can also be clicked and it can take you into another page. The navigation bar is also another interaction that users can click on. When they hover over these options, an underline will appear, signifying that it is something you can click on.


Accessibility

When it comes to accessibility, I want people with disabilities to have the same equal access everyone else has.

When it comes to font sizing, I have made it so that when people zoom in, the font would also become larger, making it easier for those that have impaired vision to read better. I have also put spacing for the words with 'line-height' so they do not look like they are all stuck together. For the images, I have put an 'alt' attribute so those that have vision impairment are able to visualize the image by the text. Different font sizes used can also help those distinguish which part they are reading.
